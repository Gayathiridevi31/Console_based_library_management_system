📚 Library Management System (Console Based – Java)
📌 Project Overview

The Library Management System is a console-based Java application that allows users to manage books and members in a library.

It supports adding books, registering members, borrowing and returning books, generating reports, and tracking top borrowers.

This project demonstrates:

Object-Oriented Programming (OOP)

Collections Framework (HashMap, ArrayList)

Java Time API (LocalDate, ChronoUnit)

Basic Business Logic Implementation

🚀 Features
📖 Book Management

Add new books

Track total copies

Track available copies

View borrowed members

👤 Member Management

Register new members

Limit members to borrow maximum 3 books

Track total books borrowed

🔄 Borrow & Return

Borrow book (if available)

Prevent duplicate borrowing of same book

Auto due date (7 days from borrow date)

Fine calculation (₹2 per overdue day)

Return book and update availability

🔎 Search Functionality

Search by book title

Search by author name

📊 Reports

Member Report (borrowed books + due dates)

Book Report (available copies + borrowers)

Top Borrowers (sorted by total borrow count)

🛠️ Technologies Used

Java

OOP Concepts

Java Collections Framework

Java Time API

Console-based UI
